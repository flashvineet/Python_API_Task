This project demonstrates how to fetch and display user data from a public API using Python, the requests library, and JSON handling.

#Features
Uses GET API (JSONPlaceholder)
Extracts and prints:

Name
Username
Email
City

Includes Bonus Filtering:
Shows only users whose city starts with S
Handles:
API request failure
Empty dataset

#Technologies Used
Python 3
Requests library
JSON handling

#Installation
1. Install Python (if not installed)
    Download from https://www.python.org

2. Install required packages
    pip install -r requirements.txt

How to Run
Open a terminal in the project folder and run:
python main.py

API Endpoint Used
https://jsonplaceholder.typicode.com/users
